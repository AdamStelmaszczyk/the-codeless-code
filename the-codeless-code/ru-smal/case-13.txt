Title: Эволюция

Послушник проанализировал код учителя Java и досконально
перенял его стиль: используемые шаблоны проектирования,
соглашения об именовании переменных, количество пробелов,
используемых для отступа каждого вложенного блока. Даже
комментарии превосходно имитировали язык учителя.

Во время следующего анализа кода послушник с удивлением
обнаружил, что учитель Java чрезвычайно недоволен.

"Но, учитель", - возразил студент. "Я следовал вашему
примеру до последней точки с запятой. Посмотрите на этот
класс, который вы написали в прошлом месяце: разве он не
похож на мой код, как два яйца из одной кладки?"

"В этом и заключается проблема", - нахмурился
учитель. "Вчера я был дурак, за неделю до этого - идиот, а
месяц назад - дебил. Не показывай мне код, который я мог бы
написать вчера. Покажи мне код, который я напишу завтра".
