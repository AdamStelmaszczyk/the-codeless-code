Title: Освобождение

Учитель Банзен изучал послушников некоего далёкого монастыря. Их разработки и реализации вызвали у него сильное беспокойство.

"Кто проверяет ваш код?" - спросил Банзен.

"Наши учителя сказали нам смотреть код друг друга", - объяснил один из послушников. "Это освобождает их для более важных дел".

"Неужто?" - сказал Банзен. Он с силой швырнул свой посох через всю комнату в электрощит монастыря, который взорвался снопом искр. Все экраны погасли.

Банзен забрал свой посох, заявив: "Сейчас их электроны тоже освобождены".
