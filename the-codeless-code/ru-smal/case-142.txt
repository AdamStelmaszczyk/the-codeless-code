Title: Слепой ведёт слепого
Illus.0.title: Имея достаточно глазных яблок, все насекомые в конце концов скажут, что наелись и не могут съесть ещё одно глазное яблоко.


<i>[[Banzen|Банзен]] собрал клан Паука и сказал:</i> //
"Учитель Суку находится слишком далеко, чтобы наблюдать за вашим следующим релизом, //
и она не оставила настоятеля, чтобы управлять вами. //
Как вы будете искать ошибки в коде?"

<i>Первый монах сказал:</i> //
"Кто может лучше знать недостатки свиньи, чем другая свинья? //
Я проверю код моего брата, и он мой. //
Мы будем глазами друг друга".

<i>Второй монах сказал:</i> //
"Свинья может познать себя и внутри, и снаружи, //
но только повар может сказать, хорошо ли она прожарена. //
Я буду доверять глазам наших тестеров, чьим священным долгом является поиск недостатков".

<i>Третий монах сказал:</i> //
"Сочная свинья может быть прожарена до совершенства, //
но повар может лишиться жизни, если Императрица хотела говядину. //
Я напишу демо-приложение для наших заказчиков и буду доверять только их глазам".

<i>Четвёртый монах сказал:</i> //
"Повар заслуживает потери головы, если он послал за заказом идиота! //
Доверие к другим - это отказ от ответственности. //
Я буду использовать это приложение сутки напролёт и буду доверять только своим глазам".

<i>Пятый монах сказал:</i> //
"Перед тем, как обслуживать столы самому, надо обратиться к доктору, если слышите плохо! //
Доверие только себе - это проявление высокомерия. //
Мои автоматизированные модульные тесты будут моими глазами".

<i>Шестой монах сказал:</i> //
"Плох тот доктор, что оценивает тело, изучая только ноги. //
Пусть пациент пройдёт несколько километров! //
Мой автоматизированный поисковый робот будет моими глазами".

<i>Седьмой монах сказал:</i> //
"Только свиной доктор оценивает тело только по тому, что можно увидеть. //
Надо собрать кровь и желчь! Мой статический анализатор кода будет моими глазами".

<i>Восьмой монах спрятал голову под накидкой и сказал:</i> //
"Семь дорог начинаются здесь, но этот слепой человек должен остаться. //
И ежели семеро придут и расскажут, что видели, то я буду видеть дальше любого из них. //
Все эти монахи будут моими глазами".

<i>Банзен сказал свой ученице Сато:</i> //
"Второй монах ничего не делает. //
Восьмой монах ничего не делает. //
Что советует Ваш третий глаз?"

<i>Сато ответила своему учителю:</i> //
"Ничего второго монаха не имеет силы, //
но ничего восьмого монаха не имеет силы в восемь раз больше. //
Восьмой монах должен стать новым настоятелем, а второй должен прислуживать ему".

Таким образом, восьмого монаха повысили,//
а второго - наказали.
