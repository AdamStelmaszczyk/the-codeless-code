Title: Компромисс

Два монаха пришли к учителю Java с очень разными подходами к реализации определённого проекта. Каждый представил свою идею страстно и убедительно.

После некоторого обсуждения, учитель сказал им разделить подсистемы. Чтобы каждый монах мог взять половину и осуществить собственный подход.

Первый монах возразил: "Разве для любой проблемы есть более одного решения, которое превосходит остальные?"

Второй монах был также недоволен: "Разве не правильно действовать последовательно и, таким образом, избежать хаоса, даже если наилучшее решение пока не ясно?"

"Да," - сказал учитель Java. "Разве никто из вас не обратил внимание?"
