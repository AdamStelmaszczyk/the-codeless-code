Auf den Kacheln eines WCs nahe Java-Meister [[Bawan]]s Büro: *

{{*}} Einige Beobachtungen des Autors: //
- Das bedeutet, dass jede Binärzahl Chromosomen beiderlei Geschlechts hat, außer Zahlen der Form (2^n)-1 mit n >= 0. //
- Es bedeutet auch, dass das bitweise Komplement (~) eine Geschlechtsumwandlung ist. //
- Es wäre natürlich, Binärzahlen das Geschlecht wie folgt zuzuweisen: Gerade Zahlen (auf 0 endend) sind weiblich, und ungerade (auf 1 endend) männlich. Ich persönlich fände allerdings verwirrend, wenn jemand das anatomische Merkmal, das mein Geschlecht definiert, niedrigstwertiges Bit nennen würde.
